Wordle Game

Описание
Это реализация популярной игры Wordle на Python. Игра загадывает пятибуквенное слово, и игроку нужно угадать его за ограниченное количество попыток. После каждой попытки буквы подсвечиваются цветами:

Зеленый: буква на правильном месте.

Желтый: буква есть в слове, но не на этом месте.

Серый: буквы нет в слове.

Репозиторий: https://github.com/itp10ppa/wordle

Правила игры в КООП
Друг загадывает случайное пятибуквенное слово.
Игрок вводит пятибуквенное слово.
После ввода подсвечиваются буквы по цветам (зеленый, желтый, серый).
У игрока есть 6 попыток, чтобы угадать слово.
Если слово угадано — победа! Если попытки закончились — проигрыш, показывается правильное слово.

Правила игры в Соло
Также есть режим игры с компьютером. Копмьютер выбирает случайное слово из списка и загадывает его пользователю.
Игрок вводит пятибуквенное слово.
После ввода подсвечиваются буквы по цветам (зеленый, желтый, серый).
У игрока есть 6 попыток, чтобы угадать слово.
Если слово угадано — победа! Если попытки закончились — проигрыш, показывается правильное слово.

Требования
Python 3.13 или выше.

Библиотеки: random (стандартная)

Установка
Клонируйте репозиторий:

git clone https://github.com/itp10ppa/wordle.git
wordle

python main.py
Игра начнется в терминале. Следуйте инструкциям на экране.

Пример игры
Компьютер загадал слово 'океан'

Попытка 1: носок
н[желтый] о[желтый] с[серый] о[серый] к[желтый]

Попытка 2:океан
о[зеленый] к[зеленый] е[зеленый] а[зеленый] н[зеленый]
